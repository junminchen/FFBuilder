# Electrolyte Bulk Simulation Example\n\nThis example demonstrates how to build and run a bulk electrolyte simulation (1M LiPF6 in EC/DMC).\n\n## Steps to Run:\n\n1. **Build the System**:\n   `bash\n   python build_electrolyte.py\n   `\n   This generates `init_system.pdb`.\n\n2. **Run NPT Simulation**:\n   `bash\n   python run_MD_bulk.py\n   `\n   This performs energy minimization, equilibration, and production run on CUDA GPU.\n\n## Output Files:\n- `init_system.pdb`: Initial configuration\n- `trajectory.dcd`: MD trajectory\n- `simulation.log`: Potential energy, density, and temperature log
