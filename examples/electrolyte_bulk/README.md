# Electrolyte Bulk Example\n\n一键构建并运行 1M LiPF6 in PC/DMC (1:1) 的电解液 NPT 模拟。\n\n## 运行步骤:\n\n1. **建模**: 生成初始 PDB 结构\n   `bash\n   python build_electrolyte.py\n   `\n\n2. **运行**: 执行能量最小化、预平衡和生产运行\n   `bash\n   python run_MD_bulk.py\n   `\n\n## 文件说明:\n- `forcefields/`: 包含 OPLS-AA 溶剂和 0.75 缩放电荷盐力场\n- `molecules/`: 溶剂分子 PDB 模板\n- `init_system.pdb`: 生成的初始体系构型\n- `simulation.log`: 能量、密度等热力学数据\n- `trajectory.dcd`: 分子轨迹文件
